# Welcome to the Y-Foundry DAO Design System

This Design System is built as a reusable component library for Y-Foundry DAO

We follow the principle of [atomic design](https://bradfrost.com/blog/post/atomic-web-design/) as much as possible and want every component built to be re usable and composable into larger storybook pages
